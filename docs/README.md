
# Лабораторная работа №5 - Вероятностные алгоритмы проверки чисел на простоту

## Описание
Реализация трех вероятностных алгоритмов проверки чисел на простоту согласно методическим указаниям лабораторной работы №5.

## Реализованные алгоритмы
- **Тест Ферма** - основан на малой теореме Ферма
- **Тест Соловэя-Штрассена** - использует символ Якоби  
- **Тест Миллера-Рабина** - наиболее надежный вероятностный тест

## Структура проекта



lab05/
├── primality_tests.py # Основные алгоритмы проверки простоты
├── test_comprehensive.py # Расширенное тестирование и анализ
├── lab05_report.md # Отчет по лабораторной работе
└── README.md # Документация



## Требования
- Python 3.6 или выше
- Стандартные библиотеки Python

```bash
# Основное тестирование алгоритмов

# Комплексное тестирование и анализ ошибок

Особенности реализации
Все алгоритмы реализованы согласно методическим указаниям
